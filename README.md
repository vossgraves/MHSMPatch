## Introduction 

Rootless implementation of LSPosed framework, integrating Xposed API by inserting dex and so into the target APK.

We sincerely invite you to join our [Telegram](https://t.me/OQPatch) group to get more information and updates about OQPatch.

## Supported Versions

- Min: Android 8.1
- Max: In theory, same with [JingMatrix/LSPosed](https://github.com/JingMatrix/LSPosed#supported-versions)

## Download

For stable releases, please go to [Github Releases page](https://github.com/7723mod/OQPatch/releases)
For canary build, please check [Github Actions](https://github.com/7723mod/OQPatch/actions)
Note: debug builds are only available in Github Actions

## Usage

+ Through jar
1. Download `oqpatch.jar`
1. Run `java -jar oqpatch.jar`

+ Through manager
1. Download and install `manager.apk` on an Android device
1. Follow the instructions of the manager app
